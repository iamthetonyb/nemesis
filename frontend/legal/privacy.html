Privacy

USA Spending Watch is designed as a public, low-tracking spending dashboard. The production frontend is static and does not require an account.

Data We Load

The app loads public dashboard files from this site, map tiles from OpenStreetMap, and optional Cloudflare Turnstile only if feedback protection is configured and opened.

Data We Do Not Collect

We do not run Google Analytics, Meta Pixel, Hotjar, LinkedIn Insight Tag, or behavioral ad trackers. We do not set cookies on first visit.

Feedback

The Feedback button opens a GitHub Issue draft. If you submit it, GitHub receives the content under GitHub's own account and privacy controls.

Public Sources

Displayed spending records come from public official sources such as USAspending.gov and government open-data portals.
